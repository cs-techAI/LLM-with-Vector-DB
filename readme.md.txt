# Markdown Conversion and LLM Interaction

This Streamlit application allows users to upload documents in various formats (.txt, .pdf, .docx), convert them to Markdown, and interact with different language models for summarization. The app also supports embedding generation using the Sentence Transformers library.

## Features

- Upload files or enter text directly.
- Convert uploaded documents to Markdown format.
- Summarize content using Hugging Face models or OpenAI's GPT-3.
- Generate embeddings for the content if the Sentence Transformers library is available.
